# E-Commerce Sales Analysis and Prediction

## Project Overview

This project presents an end-to-end machine learning workflow for analyzing e-commerce product data and building predictive models. The project includes data understanding, preprocessing, sentiment analysis, feature engineering, exploratory data analysis, handling class imbalance, model training, evaluation, explainable AI using SHAP, and product-level testing.

## Project Workflow

- Data Importing and Understanding
- Data Cleaning and Preprocessing
- Sentiment Analysis
- Feature Engineering
- Handling Class Imbalance
- Exploratory Data Analysis (EDA)
- Machine Learning Model Training and Evaluation
- Sales Classification using XGBoost
- Regression Analysis
- Model Interpretation using SHAP
- Product Filtering and Testing
- Model Saving for Deployment

## Technologies Used

- Python

## Dataset

The project uses an Amazon product reviews dataset loaded programmatically using KaggleHub.

## How to Run the Project

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required Python libraries.
3. Run the notebook cells sequentially from top to bottom.
4. The dataset will be downloaded programmatically using KaggleHub.
5. Review the generated visualizations, model evaluation results, predictions, and SHAP explanations.

## Project Notebook

The complete implementation, analysis, model training, evaluation, and explainability workflow is available in the Jupyter Notebook included in this repository.
